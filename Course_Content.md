# Python Crash Course for Machine Learning - Course Content

This repository contains a **beginner-friendly Python crash course** designed specifically for **Machine Learning learners**. Each lesson focuses on practical Python concepts with examples, exercises, and connections to ML.

---

## Lessons Overview

1. **Arithmetic and Logical Operations**  
   Learn basic math operations, logical comparisons, and how to use them in Python.

2. **Conditional Statements**  
   if, elif, else statements to control the flow of your programs.

3. **Lists in Python**  
   Creating, modifying, and iterating over lists.

4. **Tuples**  
   Immutable sequences and their uses.

5. **Strings**  
   Working with text, string methods, and formatting.

6. **Dictionaries**  
   Storing key-value pairs, accessing, updating, and iterating.

7. **For Loops**  
   Iterating over sequences with loops.

8. **Functions in Python**  
   Writing reusable code with functions, parameters, and return values.

9. **Classes in Python**  
   Introduction to object-oriented programming.

10. **NumPy Arrays**  
    Efficient numerical arrays and basic operations for ML.

11. **Pandas**  
    Working with tabular data using DataFrames and Series.

12. **Plotting and Visualization**  
    Introduction to Matplotlib for visualizing data.

13. **Seaborn**  
    Advanced visualization and statistical plots.

---

## Additional Notes

- Each lesson will be in a separate Python file (e.g., `01_arithmetic_logical.py`)  
- All datasets will be stored in the `datasets/` folder  
- This course assumes **Python 3.9+** and **VS Code** as the editor
